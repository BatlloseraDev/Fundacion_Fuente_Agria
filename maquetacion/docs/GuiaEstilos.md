# Guía de Estilos UI - Fundación Fuente Agria

_Esta guia de estilos está en proceso de creación por el equipo de desarrollo_

## 1. Filosofía de Diseño
Buscamos transmitir **inocencia, pulcritud y optimismo**. Hemos abrazado el **Blanco (#FFFFFF)** como elemento principal, simbolizando transparencia y cuidado.

## 2. Tipografía
Combinación seleccionada por su legibilidad (Accesibilidad AAA):
* **Títulos (Nunito):** Una fuente *sans-serif* redondeada. Sus terminaciones curvas reducen la agresividad visual y evocan cercanía, ideal para el contexto de la fundación.
* **Cuerpo (Open Sans):** Neutral, extremadamente legible en pantallas y tamaños pequeños.

## 3. Paleta de Color (Regla 60-30-10)
* **60% Fondo (Blanco/Neutro):** Espacios amplios en blanco para reducir la carga cognitiva.
* **30% Marca (Friendly Blue):** `#3B82F6`. Un azul vibrante pero no oscuro. Transmite confianza  y tecnológica sin ser frío.
* **10% Acción (Warm Apricot):** `#F97316`. Usado exclusivamente para botones de llamadas a la acción (CTA).

## 4. Componentes Clave

### Botones (Pill Shape)
Usamos formas de píldora para eliminar "esquinas peligrosas" visualmente.
